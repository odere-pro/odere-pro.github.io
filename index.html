    About

    From prototype to production.

    I'm a forward deployed AI engineer — I embed with the problem, take agents from prototype to production, and build the plugins, multi-agent systems, and cloud infrastructure that keep them reliable at scale. The primary user is increasingly an LLM, so I design for it: stable contracts, prose-for-agents docs, and deterministic gates that hold whether a human or an agent is driving.

    I ship in the open. The odere-pro stack spans a published npm CLI and a Claude Code plugin marketplace — claude-aws-architect alone runs 5 agents, 6 MCP servers, and 34 validation gates, grounding every claim against AWS docs. As a founder, I turn agent-native tooling into products teams depend on — moving fast without trading away reliability.
